Sliding Menu Info
On the Graph page we have three buttons and three toggles.
Load Test Data

Load Test Data allows you to load supplied test data to see how everything works. You can not export this data as it is only for learning.

Reset

Next there is a rest button which allows you to reset the chart. This will erase stored data but not if you have saved it on the export page.

Toggles

Touch a toggle and it will remove the corresponding colored line from the graph. Touch it again and it is back. This helps in viewing the data.

Start

Pressing start will start the geolocation services on your device. You must allow the device to use your location for it to work. For best results be in a clear out door area away from tall buildings as this uses satellites in space and needs a clear shot. If nothing happens you may need to change the mode to bike or drive on the sliding menu.
