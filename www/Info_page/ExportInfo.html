In Expert we have for choices.
Save data in New File

First, Save data in New File. If you have gathered data on the graph page by pushing the start button on the upper right hand corner then you may save your data to a file in your device.

Remove a File

Next there is remove file which allows the user to remove unneeded files.

Show Route in Google Maps

The third button will display a google map of the points that were gathered.

Select File and Send Email

The forth button, Select and Email allows you to email a CSV file of your data points along with the google map image.
